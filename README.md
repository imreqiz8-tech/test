# Авто-поднятие резюме hh.ru

Расширение для Chrome / Edge / любого Chromium-браузера (Manifest V3),
которое **автоматически поднимает ваше резюме на hh.ru каждые 4 часа** —
ровно с тем интервалом, который разрешает hh.ru.

## Как это работает

- Сервис-воркер (`background.js`) ставит alarm `chrome.alarms` с периодом
  240 минут. По срабатыванию он находит вкладку с резюме (или открывает
  её в фоне) и обновляет страницу.
- Контент-скрипт (`content.js`) на странице ищет кнопку
  «Поднять в поиске» (`data-qa="resume-update-button_actions"` и
  альтернативные селекторы) и кликает по ней.
- Результат (успех / кулдаун / ошибка) отправляется обратно в фон,
  сохраняется в `chrome.storage.local` и показывается в попапе и
  системном уведомлении.

## Установка (режим разработчика)

1. Откройте `chrome://extensions` (или `edge://extensions`).
2. Включите **Режим разработчика**.
3. Нажмите **Загрузить распакованное** и выберите эту папку.
4. Залогиньтесь на [hh.ru](https://hh.ru) под своим аккаунтом.

Готово — расширение само откроет страницу резюме в фоне и нажмёт
«Поднять в поиске», как только пройдут очередные 4 часа.

## Управление

Кликните по иконке расширения, чтобы:

- включить / выключить автоматическое поднятие;
- увидеть время следующего и последнего запуска;
- поднять резюме вручную («Поднять сейчас»);
- открыть страницу со списком резюме.

## Структура

| Файл           | Назначение                                              |
| -------------- | ------------------------------------------------------- |
| `manifest.json`| Манифест MV3, права, content scripts                    |
| `background.js`| Сервис-воркер: alarms, оркестрация, уведомления         |
| `content.js`   | Поиск и клик по кнопке «Поднять в поиске» на hh.ru      |
| `popup.html`   | UI попапа                                               |
| `popup.js`     | Логика попапа: переключатель, статус, ручной запуск     |

## Замечания

- hh.ru ограничивает поднятие интервалом ~4 часа. Если резюме ещё в
  кулдауне, расширение это распознает и попробует на следующем тике.
- Расширение не хранит и не отправляет никаких данных наружу —
  всё работает локально в вашем браузере.
- Иконки `icon128.png` опциональны; без них уведомления просто
  будут без картинки.
