# 🎵 Мелодия — Интерактивный музыкальный плеер

Одностраничный музыкальный плеер на чистом HTML/CSS/JS, использующий
Web Audio API для синтеза звука прямо в браузере. Без зависимостей,
без сборки — просто откройте `index.html`.

## Возможности

- **8 классических мелодий** — Бетховен (Ода к радости, К Элизе, Лунная соната),
  Моцарт, Бах, Пахельбель, Twinkle Twinkle, Happy Birthday
- **6 инструментов/тембров** — пианино, флейта, труба, 8-bit, гитара, орган
  (через комбинацию осцилляторов и гармоник)
- **Реал-тайм визуализатор** на основе `AnalyserNode` (FFT-спектр)
- **Регулировка темпа и громкости** ползунками
- **Прогресс-бар** с отображением времени
- **Подсветка текущей ноты** с анимацией
- **Светлая/тёмная тема** с сохранением в `localStorage`
- **Управление с клавиатуры**: `Space`, `S`, `←`/`→`, `T`
- **Glassmorphism**-дизайн, анимированный градиент, плавающие фоновые ноты
- **Адаптивная вёрстка** для мобильных
- **Поддержка `prefers-reduced-motion`** для пользователей с чувствительностью к анимации
- **Семантическая разметка**, ARIA-атрибуты, `aria-live` статус

## Запуск

Откройте `index.html` в любом современном браузере. Никакой
установки или сервера не требуется.

## Технологии

- Web Audio API (`OscillatorNode`, `GainNode`, `AnalyserNode`)
- CSS: custom properties, backdrop-filter, CSS-сетка, анимации
- Vanilla JS (ES2020+), без фреймворков и сборщиков
